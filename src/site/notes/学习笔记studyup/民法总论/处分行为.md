# 法律效果
变动既存权利，不发生新债权债务
- 必定导致一方（处分人）既存权利的减损
- 可能导致另一方（取得人）既存财产的增加
- 可能不导致任何一方的财产增加：抛弃

# ❓疑难问题
## 界定
- 形成行为：非处分行为
	- 处分行为是双方法律行为，需要双方之合意方得成立
	- 形成行为的特点是使相对人处于他决地位
- 对他人处分之同意
	- 权利人之同意：处分行为
	- 非权利人之同意（如国家审批）：非处分行为
- 死因处分：遗嘱既非负担也非处分行为
- 预告登记：非处分行为
- 租赁物之交付：非处分行为
	- 租赁物之交付未导致既存权利的变动或减损
## 冒名处分≠借名
先要通过意思表示解释，确定相对人是否注重名义人
- 不注重名义人：构成自己行为，以无权处分、善意取得处理
- 注重的是名义人：构成他人行为，🗣️类推无权代理、表见代理处理
