**第二百八十八条**　不动产的相邻权利人应当按照有利生产、方便生活、团结互助、公平合理的原则，正确处理相邻关系。 

</div></div>


相邻关系：法律为调和相邻不动产的利用，在其所有人或者其他权利人之间所设定的权利义务关系 #背诵/概念 
- 目的：实现相邻不动产权利人之间的利益平衡 + 充分发挥物的效用
- 性质：不动产所有权的法定限制

## [[学习笔记studyup/民法总论/容忍义务#相邻关系\|容忍义务#相邻关系]]
# 规制方式：规则→习惯→原则
- 相邻关系的复杂性
## 原则
### 有利生产：物尽其用
### 方便生活：权力行使不能给邻人带来过度不便
### 团结互助：调解优先
### 公平合理：权利保护与限制的个案平衡
## 当地习惯
>民第 289 条：
<div class="transclusion internal-embed is-loaded"><a class="markdown-embed-link" href="/////#t289" aria-label="Open link"><svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="svg-icon lucide-link"><path d="M10 13a5 5 0 0 0 7.54.54l3-3a5 5 0 0 0-7.07-7.07l-1.72 1.71"></path><path d="M14 11a5 5 0 0 0-7.54-.54l-3 3a5 5 0 0 0 7.07 7.07l1.71-1.71"></path></svg></a><div class="markdown-embed">



**第二百八十九条**　法律、法规对处理相邻关系有规定的，依照其规定；法律、法规没有规定的，可以按照当地习惯。 

</div></div>

- 适用限制：法律法规没有规定的
	- 先于原则后于法律法规
- 适用方法：当事人举证，法院依职权查明
# 构成要件：不动产的相邻权利人
## 不动产：土地，附着于土地的建筑物
>民第 292 条：
<div class="transclusion internal-embed is-loaded"><a class="markdown-embed-link" href="/////#t292" aria-label="Open link"><svg xmlns="http://www.w3.org/2000/svg" width="24" height="24" viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2" stroke-linecap="round" stroke-linejoin="round" class="svg-icon lucide-link"><path d="M10 13a5 5 0 0 0 7.54.54l3-3a5 5 0 0 0-7.07-7.07l-1.72 1.71"></path><path d="M14 11a5 5 0 0 0-7.54-.54l-3 3a5 5 0 0 0 7.07 7.07l1.71-1.71"></path></svg></a><div class="markdown-embed">



**第二百九十二条**　不动产权利人因建造、修缮建筑物以及铺设电线、电缆、水管、暖气和燃气管线等必须利用相邻土地、建筑物的，该土地、建筑物的权利人应当提供必要的便利。 

</div></div>

## 相邻
- ✅毗邻
- ✅实质判断：能产生影响力的特定地理环境（环境污染）
## 权利人
- 不动产所有人、用益物权人（居住权人）、占有人
# 内容：权利义务
1. 不作为义务
2. 容忍义务
3. 损害避免义务
# 类型
